jAM by fredda

automatic music transcription application written in java

This application implements a prototype for automatic music transcription. It was part of my bachelor thesis. Two PitchDetecttionAlgorithms are in use: Yin (http://www.ircam.fr/pcm/cheveign/pss/2002_JASA_YIN.pdf) MPM (http://miracle.otago.ac.nz/tartini/papers/A_Smarter_Way_to_Find_Pitch.pdf) The implementation of those can be found at https://github.com/JorenSix/TarsosDSP It uses abc4j to show a musical score, save to image, save MIDI etc..

Dependencies

abc4j and tarsos (source included)
